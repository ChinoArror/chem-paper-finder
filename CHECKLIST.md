# 功能测试 Checklist

- [ ] 未登录访问首页会进入 Auth Center 登录入口。
- [ ] DOI 输入能进入队列，完成后显示候选文献。
- [ ] 完整期刊页码输入能返回多个候选。
- [ ] 作者 + 期刊 + 年份 + 关键词输入能返回模糊候选。
- [ ] 候选显示标题、作者、期刊、年份、卷期页、DOI、置信度和匹配理由。
- [ ] 开放 PDF 候选显示来源、版本类型、license、保存状态。
- [ ] 点击开放 PDF 下载不会刷新页面，浏览器直接触发保存。
- [ ] 无开放 PDF 时可生成并下载题录 PDF。
- [ ] 有开放 PDF 时不会自动生成题录 PDF，点击后才生成。
- [ ] 历史列表和详情页可打开、删除整条记录、删除单个候选。
- [ ] 已保存开放 PDF / 题录 PDF 可删除。
- [ ] 移动端无横向溢出，主要按钮可点击。
- [ ] 生产 `/api/health` 返回 `{ "ok": true }`。
